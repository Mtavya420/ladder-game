## 프로젝트 핵심 목표

- 플레이어 수에 따른 랜덤 프로필과 랜덤 사다리 설정 기능
- 사다리 게임 로직을 리액트의 상태 관리와 자바스크립트로 구현
- 캔버스를 활용하여 사다리 경로를 그리게 하는 기능
